# 前言
课程（自然语言处理与文本挖掘）关于文本分类的代码。
一共6个模型。
1. RidgeClassifier
2. SVM
3. fastText
4. TextCNN
5. TextRNN
6. BERT

代码全部是jupyter notebook形式。训练环境：Windows 10 专业版21H1，CPU Intel(R) Core(TM) i7-9700K CPU @ 3.60GHz，内存32GB，显卡NVIDIA GeForce RTX 2070，cuda版本10.1.
